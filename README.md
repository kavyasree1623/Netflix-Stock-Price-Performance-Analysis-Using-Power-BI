📊 Netflix Stock Price Performance Dashboard (Power BI)
📌 Project Overview

This project focuses on analyzing historical Netflix stock price performance using Microsoft Power BI.
The dashboard provides interactive visual insights into stock trends, price fluctuations, and trading volume over time.

The main objective of this project is to understand stock market behavior using Business Intelligence and Data Visualization techniques.

🎯 Objectives

Analyze Netflix stock price trends over multiple years

Identify maximum, minimum, and average closing prices

Study trading volume fluctuations

Enable dynamic filtering using slicers for better decision-making

Build an interactive and user-friendly Power BI dashboard

📂 Dataset

Source: Historical Netflix Stock CSV Dataset

Features Used:

Date

Open Price

Close Price

High Price

Low Price

Volume

⚙️ Tools & Technologies Used

Microsoft Power BI Desktop

Power Query Editor

Data Modeling

DAX Calculations

Data Visualization Techniques

📈 Dashboard Visualizations

The dashboard includes the following visuals:

KPI Cards (Average Close Price, Maximum Price, Minimum Price, Total Volume)

Line Chart showing Netflix Stock Trend

Column Chart showing Monthly Average Close Price

Area Chart showing Trading Volume Distribution

High vs Low Price Comparison

Year Slicer for Interactive Filtering

🧹 Data Preprocessing

Converted Date column to proper Date format

Created new calculated columns such as Year and Month

Removed inconsistencies and prepared dataset using Power Query

📊 Key Insights

Netflix stock shows strong growth trends during certain periods

Significant volatility is observed during market fluctuations

Trading volume spikes indicate increased investor activity

Monthly average prices help understand seasonal market behavior
